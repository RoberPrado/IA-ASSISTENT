📢 Cómo dictar correctamente

Para que la IA genere correctamente el requerimiento, debes usar la siguiente estructura:

  • Como [rol del usuario]
  • Quiero [qué deseas que haga el sistema]
  • Para [beneficio u objetivo]

📌 Ejemplo:
"Como supervisor de almacén quiero que el sistema genere un código de barras para cada nuevo producto para evitar errores de inventario."
